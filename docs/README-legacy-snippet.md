# 🏁 CHRONIT Racing System

Sistema profesional de cronometraje para karts y deportes de motor.

## 📋 Requisitos

- **Docker** (instalación automática con el script)
- **ESL-400 Decoder** conectado por USB
- **Linux, Windows o macOS**

## 🚀 Instalación (1 minuto)

### Linux / macOS
```bash
chmod +x setup.sh
./setup.sh
./start.sh
